# Мой сайт

Это мой сайт с 6 страницами.  
Все страницы связаны ссылками между собой, есть стили и изображения.  

## Структура проекта

- index.html — главная страница  
- page2.html — страница 2  
- page3.html — страница 3  
- page4.html — страница 4  
- page5.html — страница 5  
- page6.html — страница 6  
- style.css — стили сайта  
- script.js — скрипты сайта  
- images/ — папка с изображениями  

## Как открыть сайт

Открой файл `index.html` в браузере или используйте GitHub Pages:
https://avisales.github.io/mysite/
